# Networking & Reconnaissance

## Objective
The objective of this task is to understand basic networking concepts such as IP addresses, DNS, ports, and protocols, and to perform simple reconnaissance using command-line tools to gather publicly available information.

## Networking Concepts

### 1. IP Address
An IP address is a unique address assigned to a device on a network. It helps identify devices and allows communication between them.

### 2. DNS
DNS stands for Domain Name System. It converts domain names such as `example.com` into IP addresses.

### 3. Ports
Ports are communication endpoints used by services and applications. Some common ports are:
- 80 → HTTP
- 443 → HTTPS
- 53 → DNS
- 21 → FTP
- 22 → SSH

### 4. Protocols
Protocols are rules used for communication between devices. Examples:
- TCP
- UDP
- HTTP
- HTTPS
- DNS
- ICMP
